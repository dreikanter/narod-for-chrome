// Short description in English

Download files from Yandex.Narod (narod.yandex.ru) without typing all these annoying CAPTCHAs.

// Detailed description

В файловом хостинге Яндекса все хорошо, кроме занудной необходимости доказывать каждый раз, что ты не злой человекоподобный робот. В других браузерах эта проблема решается установкой Яндекс.Бара. Но версии Бара для Google Chrome сейчас нет и, похоже, в обозримом будущем ее появление не предвидится. Данное расширение решает проблему: при открытии страницы с капчей на Я.Диске в адресной строке появляется иконка, клик по которой запускает скачивание.

PS: Расширение предназначено для людей. Если вы робот, не пользуйтесь им, пожалуйста. Лучше согните что-нибудь.

Добавлены новые скриншоты и иконки, соответствующие обновлённым гайдлайнам Google.
